# Afet Yönetim Sistemi

Mobil tabanlı Afet Yönetim Sistemi, afet anında yöneticiler ve gönüllüler arasındaki koordinasyonu güçlendirmek amacıyla Flutter ve Firebase kullanılarak geliştirilmiştir.

## Proje Amacı

Bu uygulama;

- Afet bölgelerinin oluşturulmasını,
- Yakındaki gönüllülerin belirlenmesini,
- Anlık bildirim gönderilmesini,
- Görev yönetimini,
- Acil yardım taleplerini,
- Kullanıcı eğitimlerini,
- Konum tabanlı afet koordinasyonunu

tek bir mobil uygulama üzerinden gerçekleştirmektedir.

---

## Kullanılan Teknolojiler

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Cloud Messaging (FCM)
- Firebase Storage
- Firebase Functions
- Google Maps API
